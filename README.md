# SmartStudy – Bootstrap 5.3 Responsive Learning Platform

## 📌 Project Overview

**SmartStudy** is a fully responsive educational website developed using **Bootstrap 5.3** along with custom CSS styling.
The objective of this project was to explore all major Bootstrap components, utilities, and layout systems and creatively combine them into a single cohesive webpage.

This project simulates an online learning platform where students can explore courses, view pricing plans, check project galleries, and register for training programs.

The webpage is designed to work smoothly on **mobile, tablet, and desktop devices**.

---

## 🎯 Purpose of the Assignment

This project was created as part of the **Advanced Bootstrap Module Practical Exercise**.
The aim was not only to study Bootstrap examples but to practically implement them together in one integrated real-world webpage.

## 💡 Bootstrap Components Implemented

### 1. Responsive Navbar

* Sticky top navigation bar
* Dropdown menu for courses
* Integrated search bar
* Mobile responsive collapse menu

### 2. Offcanvas Sidebar

* Student dashboard panel
* Opens from the left side
* Contains navigation options like Dashboard, Courses, Assignments and Settings

### 3. Carousel (Hero Section)

* 3 slides with custom captions
* Call-to-action buttons
* Smooth fade animation
* Used as the landing banner of the website

### 4. Alerts

Different alert messages implemented:

* Info (new course announcement)
* Warning (limited seats)
* Danger (last date reminder)

### 5. Pricing Table

* Built using Bootstrap cards and grid system
* Three plans: Basic, Pro, Premium
* Badges and highlighted featured plan
* Equal height responsive layout

### 6. Modal

* Triggered from pricing section
* Displays course details dynamically

### 7. Grid Gallery

* Student project showcase
* Responsive layout using Bootstrap grid
* Hover zoom effect using custom CSS

### 8. Forms with Validation

Registration form includes:

* Text input
* Email input
* Phone number field
* Radio buttons (Online/Offline)
* Checkbox agreement
* JavaScript validation using Bootstrap validation classes

### 9. Buttons & Badges

* Multiple button styles
* Rounded buttons
* Pricing badges (Free, Popular)

### 10. Utilities Used

* Spacing (margin & padding utilities)
* Flexbox utilities
* Shadows
* Text alignment
* Colors
* Responsive classes

### 11. Sticky Footer

* Footer stays at bottom using flex utilities
* Displays project credits
